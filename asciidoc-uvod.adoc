== Sažetak
Ovaj dokument objašnjava **osnovne principe AsciiDoc **. Namijenjen je studentima koji već poznaju Markdown, ali žele naučiti alat koji nudi više mogućnosti za strukturiranje i oblikovanje akademskog i tehničkog teksta.

== Što je AsciiDoc
AsciiDoc je *plain-text* format za pisanje strukturiranih dokumenata. Tekst je lako čitljiv u izvornom obliku i može se pretvoriti u *HTML*, *PDF* ili *DOCX* pomoću alata *Asciidoctor*.

=== Glavne prednosti u oddnosu na Markdown
- podržava naslove svih razina (``=``, ``==``, ``===``)
- automatski izrađuje sadržaj i numerira poglavlja
- koristi *atribute* i *varijable*
- jednostavno dodaje __fusnote__, __tablice__, __citate__ i __blokove koda__

== Struktura dokumenta
Na početku se obično nalazi naslov, autor, datum i osnovni atributi.
Naslovi se označavaju simbolima jednakosti (``=``).
Više znakova znači nižu razinu naslova.


```adoc
Autor <email@domena.com>
:revdate: 2025-11-06
:doctype: article
:toc:
:sectnums: 
```

== Formatiranje teksta
Tekst se može podebljati, nagnuti ili precrtati.


```adoc
*podebljano*, __kurziv__, 
[line_through]#Precrtano#


```

== Popisi
Numerirani popisi koriste brojeve ili točke.
Nenumerirani popisi koriste zvjezdice ili crtice.

```adoc
. Numerirani popis
.. stavka 1
... stavka 2

```


```adoc
* Nenumerirani popis
** Stavka
*** Podstavka
```

== Citati i fusnote
Citati se prikazuju u posebnom bloku, uz ime autora. footnote:[Navodi se koriste za izjave vaznih osoba.]

[quote, Alan Kay]


____
Najbolji nacin da predvidis buducnost je da je predvidis
____


Fusnote se dodaju na kraju rečenice i sadrže kratko objašnjenje. footnote:[Ovo je objasnjenje prethodnog teksta.]



== Tablice
Tablice omogućuju pregledno prikazivanje podataka u redovima i stupcima.
Svaka ćelija odvaja se vertikalnom crtom (|).

[cols="1,2", options="header"]
|===
| Element | Opis
| Naslov | Definira galvni naslov dokumenta
| ``:toc:`` | Dodaje sadrzaj na pocetku dokumenta
| ``:sectnums:`` | Numerira poglavlja automatski 
|===

== Kod
Blokovi koda koriste se za prikaz primjera u programskim jezicima.
Mogu imati i oznaku jezika, npr. python ili bash.

[source, python]
----
def pozdrav(ime):
    print(f"pozdrav, {ime}!")
pozdrav("Studenti")
----

== Napomene
Posebni blokovi označavaju važne dijelove teksta, poput upozorenja ili savjeta.
Primjeri su: TIP, NOTE, WARNING i IMPORTANT.

[TIP]
====
Koristite Asciidoc kada zelite pisati profesionalno uredjenu dokumentaciju
====
[WARNING]
====
Pripazite da se u istom dokumentu ne mijesa `markdown` i `Asciidoc`
====
